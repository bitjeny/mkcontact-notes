# mkcontact-notes

这是一个用于归档和发布独立 HTML 页面的仓库。仓库内收录了多个不同主题的静态页面，供浏览、参考或直接使用。

## 项目简介

本仓库旨在收集和整理多个独立的 HTML 页面。这些页面各自独立，不依赖于任何特定域名或服务。每个页面均可直接打开，适合作为内容存档、学习参考或轻量级展示。

## 目录说明

仓库根目录下存放各个 HTML 页面文件。部分页面可能附带相关资源（如样式表、图片或脚本），这些资源会放在对应页面同名的文件夹中。

- `*.html` — 独立页面文件，可直接在浏览器中打开查看。
- `assets/` — 部分页面使用的公共资源目录（如通用样式、字体等）。
- `pages/` — 如果页面数量较多，会按主题或时间分文件夹组织。

## 页面归档说明

收录的页面以 HTML 格式存档，每个页面均保持原始结构。归档时不修改页面核心内容，但可能会为了兼容性进行少量调整（如修复链接、统一字符编码等）。

页面内容来源多样，本仓库仅负责存档与发布，不代表认可或背书页面中的任何观点。

## 维护说明

- 本仓库不定期更新，新增或更新页面会通过提交记录体现。
- 欢迎提交 Issue 或 Pull Request 来报告页面问题、建议改进或提供新的 HTML 页面存档。
- 提交内容请确保不违反相关法律法规，且不包含恶意代码、广告或钓鱼链接。
- 维护者会审核提交内容，并保留是否合并的最终决定权。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在浏览器中打开任意 `.html` 文件即可浏览。
3. 也可通过 GitHub Pages 在线访问（需自行配置）。
